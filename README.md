# 🗳️ SIR Roll Checker

**Naam dhundo, bina kahin upload kiye**

SIR Roll Checker ek simple browser-based tool hai jisse aap Voter List (PDF) me apna **Naam**, **EPIC Number**, ya **Serial Number** aasani se search kar sakte hain.

Is tool me PDF browser ke andar hi process hoti hai, isliye aapki file kisi server par upload nahi hoti.

---

## ✨ Features

* 🔍 Naam ya EPIC Number se search
* 📄 Text aur Scanned PDF dono support
* 🌙 Dark Mode
* 📱 Mobile Friendly Design
* 📂 Drag & Drop PDF Upload
* 🔒 Browser me hi processing

---

## 🚀 Kaise Use Kare

### 1. PDF Upload Kare

Apni Draft ya Final Voter List PDF select kare ya drag & drop kare.

### 2. Processing Ka Wait Kare

Tool PDF ko process karega. Agar PDF scanned hai to thoda extra time lag sakta hai.

### 3. Search Kare

Search box me apna Naam, EPIC Number ya Serial Number likhe.

### 4. Result Dekhe

Matching result ke saath page number aur preview dikhaya jayega.

---

## ⚠️ Important

Ye ek independent awareness tool hai.

Final verification ke liye hamesha official Election Commission records ko hi refer kare.
