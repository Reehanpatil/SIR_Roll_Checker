Ye kaam kaise karta hai, aur data kahan jaata hai?

1. Sab kuch local hai

PDF tumhare browser mein hi khulta hai aur process hota hai. Koi file kisi server pe upload nahi hoti.

2. Text ya scan, dono chalega

Agar PDF mein text already selectable hai, seconds mein ready ho jaata hai. Scanned/image PDF ho to OCR (Kannada+English) thoda time lega — page count ke hisaab se.

3. Result sirf ek starting point hai

Ye tool sirf khojne mein madad karta hai. Final confirmation ke liye hamesha voters.eci.gov.in pe official status check karo.

Ye ek independent jaagrukta tool hai, sarkari portal nahi. OCR results 100% accurate nahi hote — humesha official source se confirm karo.
